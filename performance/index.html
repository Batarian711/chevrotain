JavaScript Parsing Libraries Benchmark

A JSON Parser was implemented using each of the Parsing Libraries (Generators/Combinators/DSLs).

  1. Chevrotain Parsing DSL implementation

  2. Jison parser Generator grammar generated code

  3. ANTLR4 Parser Generator grammar generated code

  4. PEG.js Parser Generator grammar generated code

  5. Parsimmon Parser Combinator implementation

  6. Nearley Parser Generator grammar generated code hand built lexer

The JSON Grammar was used because it is simple to implement, simple to compare and is often already available as a sample grammar for most Parsing Tools

The input sample for the benchmark is a ~1,000 lines JSON file generated by json-generator.

All the parsers have no output(build no AST), they only parse the text (without errors).

Benchmarking is done using the benchmark.js library. The entire code for this benchmark can be found in this github repository.

Please be patient as this benchmark can take a while.
